Cara Kerja Responsif

Layout ini menggunakan beberapa teknik untuk memastikan tampilan bagus di semua perangkat:

  • Grid auto-fit dengan minmax untuk kolom yang fleksibel
  • Flexbox dengan flex-wrap untuk navigation
  • Media queries untuk breakpoint di 768px dan 480px
  • Max-width container untuk membatasi lebar konten
